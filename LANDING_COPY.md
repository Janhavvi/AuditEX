## FAQ

### Does AuditEX connect to billing systems?

No. The MVP uses manual input so teams can get a fast audit without granting finance, card, or vendor access.

### Will it tell us to cancel every AI tool?

No. The audit is intentionally conservative. If your stack is already efficient, the results page says so and offers notification capture instead of manufacturing savings.

### What tools does it support?

AuditEX supports ChatGPT, Claude, Cursor, GitHub Copilot, Gemini, OpenAI API direct, Anthropic API direct, NVIDIA NIM, Windsurf, and v0.

### Is the AI summary used for the math?

No. The savings calculations are deterministic rules. AI is used only to turn the completed audit into a concise executive summary.

### What happens when an audit shows high savings?

If estimated savings are over $500/month, the report surfaces Credex as the next step for capturing more of that savings through vendor and workflow review.
